SONICS DAYZ EDITOR
=================

HOW TO RUN (LOCAL):
1. Unzip files
2. Double click index.html
3. Paste types.xml or events.xml
4. Use buttons to modify
5. Save output

DEPLOY TO CLOUDFLARE PAGES:
1. Upload contents to GitHub
2. Create Cloudflare Pages project
3. Set build to NONE
4. Done

No Node.js required
